# jhw_ValuationLab

가치투자 분석 도구 모음. 능력범위 → 프리미엄 → 밸류에이션 → 배분 4단계를 하나의 사이트로 묶었습니다.

## 파일 구성 (단일 폴더)

| 파일 | 내용 |
|---|---|
| `index_main.html` | 허브 (네 도구로 가는 관문) |
| `index_circle.html` | 능력범위 체크리스트 |
| `index_premium.html` | 기업 프리미엄 평가 체크리스트 |
| `index_roe.html` | 밸류에이션 4-모드 분석기 |
| `index_kelly.html` | 켈리 포트폴리오 배분기 |

## ⚠️ 업로드 후 반드시 할 일

GitHub Pages는 `index.html`을 첫 화면으로 인식하므로, **`index_main.html`을 `index.html`로 rename** 하세요. 나머지 파일은 그대로 두면 됩니다(허브 링크가 그 이름들을 가리킴).

## 네 도구

1. **능력범위 체크리스트** — 거장 검증 6개 항목(린치·버핏·멍거·그레이엄·피셔·포퍼/소로스).
2. **기업 프리미엄 체크리스트** — 11개 프리미엄 요소 + 3×3 매트릭스.
3. **밸류에이션 4-모드 분석기** — PBR·ROE·N·r 중 3개로 나머지 1개 역산.
4. **켈리 포트폴리오 배분기** — 종목별 풀 켈리 값(f = p/a − q/b)을 상대 매력도로 삼아 100% 정규화 배분. 하프 켈리 안전마진 포함.

## GitHub Pages

**Settings → Pages** → Branch `main` / `/ (root)` → Save → 허브를 `index.html`로 rename.

접속: `https://salt99.github.io/jhw_ValuationLab/`

## 주의

- 모든 데이터는 `localStorage`(본인 기기에만 저장, 서버 전송 없음). 네 도구는 서로 다른 저장 키를 써서 충돌하지 않습니다.
- 켈리 배분기는 확률·목표가가 정확하다고 가정합니다. 실제로는 추정치이므로 하프 켈리 이하를 권장하며, 매매 지시가 아닌 사고 보조 도구입니다.
